Additional resources

Tools and learning materials for assistive technology design

A curated set of CAD, electronics, documentation, and assistive technology resources used by Beaver Works AT students and alumni.

CAD / 3D Modeling
  • Onshape — Cloud-based CAD with collaboration and free academic access.
  • Fusion 360 — Autodesk CAD, engineering, electronics, and manufacturing software; free academic license.
  • Tinkercad — Beginner-friendly browser-based 3D modeling for new CAD users.
  • FreeCAD — Open-source parametric modeling software for mechanical engineering and product design.
  • Blender — Free 3D creation suite for modeling, rendering, and animation.
  • Keyshot 3D — Rendering and animation software for photorealistic visuals.
Electronics & programming
  • Arduino — Open-source electronics platform with a free IDE for prototypes.
  • Raspberry Pi — Low-cost single-board computer for programming and electronics projects.
  • MIT App Inventor — Browser-based app builder from MIT for mobile apps without code.
  • Scratch — Visual programming tool from MIT Media Lab for learning coding basics.
Circuit design
  • Tinkercad Circuits — Browser-based circuit simulator integrated with Tinkercad.
  • Fritzing — Open-source EDA software for prototyping circuits.
  • KiCad — Free PCB design suite for electronics layouts.
Documentation & collaboration
  • Inkscape — Free vector editor for diagrams and technical illustrations.
  • GIMP — Open-source image editor for documentation and visuals.
  • OBS Studio — Recording and streaming software useful for documenting projects.
