Additional resources

Tools and learning materials for assistive technology design

A curated set of tools, platforms, and learning resources used across the CRE[AT]E Challenge — covering CAD, electronics, software, AI, accessibility, and more.

CAD & 3D modeling
  • Onshape — Cloud-based CAD with real-time collaboration; free academic license. The baseline CAD tool for BWSI AT.
  • Onshape Learning Center — Self-paced courses: Navigating Onshape, Introduction to Sketching, and Part Design Using Part Studios.
  • Fusion 360 — Autodesk's integrated CAD, engineering, electronics, and manufacturing suite; free for students and educators.
  • Tinkercad — Browser-based 3D modeling for beginners; completely free, no installation required.
  • FreeCAD — Open-source parametric modeler for mechanical engineering and product design.
  • Blender — Free and open-source 3D creation suite covering modeling, rendering, and animation.
  • Keyshot 3D — Professional rendering and animation software for photorealistic product visuals.
Electronics & physical making
  • Arduino — Open-source microcontroller platform with a free IDE; the most common starting point for interactive electronics prototypes.
  • Raspberry Pi — Low-cost single-board computer with GPIO pins for interfacing with sensors and actuators; far more powerful than a typical microcontroller.
  • Adafruit — Electronics retailer; Feather, Lilypad, and Bluefruit boards are designed for wearable applications. The Adafruit Learning Center has tutorials for every product.
  • SparkFun — Electronics retailer with an extensive library of beginner-friendly tutorials at learn.sparkfun.com.
  • Khan Academy — Electrical Engineering — Free courses covering circuit analysis, semiconductors, and signals. A solid foundation before tackling hardware projects.
  • Tinkercad Circuits — Free browser-based circuit and Arduino simulator. Recommended: simulate first, buy hardware second.
  • Fritzing — Open-source electronics design tool for prototyping and documenting circuits.
  • KiCad — Free and open-source PCB design suite for electronics layouts.
  • AFFOA Wearables Curriculum — Engineering and design curriculum from the Advanced Functional Fabrics of America, focused on wearable technology and functional fabrics.
Software & development
  • GitHub — Version control and code collaboration. GitHub Pages can host a project website for free.
  • GitHub Student Developer Pack — Free access to professional developer tools — hosting, domains, and more — for verified students.
  • Figma — Browser-based tool for wireframing and prototyping app and website interfaces. Free for students and educators.
  • MIT App Inventor — Visual drag-and-drop programming for building Android and iOS apps without prior coding experience.
  • Scratch — Visual block-based programming from MIT Media Lab; good for learning programming fundamentals.
  • Android Developers — Official guide to Android app development using Java or Kotlin.
  • CodingWithMitch — Video tutorials and example projects for Android development.
  • ThingSpeak — IoT analytics platform for collecting, visualizing, and acting on data from connected sensors and devices.
AI & machine learning
  • HuggingFace — The main hub for pre-trained ML models and datasets. Start here before training from scratch; see the documentation to get started.
  • Teachable Machine — Google's graphical tool for training image, sound, or pose classification models without writing any code.
  • CVAT — Online platform for annotating images and videos to create labeled datasets for machine learning.
Accessibility tools
  • randoma11y — Generates accessible color palette combinations with built-in contrast checking.
  • accessiBe accessScan — Scans a website URL and reports WCAG compliance issues.
  • Firefox Accessibility Inspector — Built-in browser tool with a colorblindness simulator and accessibility tree viewer. Open DevTools → Accessibility tab; no installation needed.
Documentation & media
  • Inkscape — Free and open-source vector graphics editor for diagrams, technical illustrations, and design files.
  • GIMP — Free and open-source image editor for photo editing and documentation visuals.
  • OBS Studio — Free recording and streaming software for capturing project demonstrations.
