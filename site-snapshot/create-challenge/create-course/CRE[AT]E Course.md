# CRE[AT]E Course

Index page for the CRE[AT]E Course — the self-paced online curriculum that teaches student teams the skills and processes they need to build assistive technology with a co-designer.

Teams work through five modules alongside their project. Registered coaches also receive classroom activity guides.

## Page structure

1. **Overview** — describes the curriculum and coach guides
2. **How it works** — four feature cards: self-paced, co-design focused, hands-on skills, classroom guides
3. **Five module cards** — numbered, each linked to its sub-page URL, with descriptions
4. **Maker Skills detail** — callout cards for all four sub-sections with individual links
5. **Interest band** — CTAs to start at Introduction or jump to Design Processes
