# Projects — Design for Accessible Educational Aids

Materials for projects created by students during the **Design for Educational Aids** spring 2021 crash course. Materials include descriptions, lesson plans, and 3D-printable files. Collection is incomplete since it was the first attempt to gather an online repository, assembled after the course.

## Student Projects

Each project links to a GitHub Pages site ([beaver-works-assistive-technology.github.io/aea_projects/](https://beaver-works-assistive-technology.github.io/aea_projects/)) with full description, lesson plan, and files.
